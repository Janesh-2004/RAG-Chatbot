# Screenshots

This directory contains screenshots and demo images for the RAG Chatbot project.

## How to Add Screenshots

1. Take screenshots of key features
2. Save them with descriptive names:
   - `chat-interface.png` - Main chat interface
   - `document-upload.png` - Document upload process
   - `multi-chat.png` - Multiple chat management
   - `message-rendering.png` - Markdown and code rendering
   - `mobile-view.png` - Mobile responsive design

3. Optimize images (recommended tools):
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   
4. Recommended dimensions:
   - Full UI: 1920x1080
   - Feature demos: 1200x800
   - Mobile: 375x812
